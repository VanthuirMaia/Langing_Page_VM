## 💻 Como Rodar Localmente

1. Clone este repositório:

```bash
git clone https://github.com/VanthuirMaia/Langing_Page_VM.git
cd landing_page_vm
```

2. Instale as dependências:

```bash
npm install
```

3. Execute o servidor de desenvolvimento:

```bash
npm run dev
```

4. Abra no navegador:

```
http://localhost:3000
```
